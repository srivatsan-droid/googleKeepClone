📝 Google Keep Clone (Vanilla JS Edition)
A minimalist Google Keep clone built with 💛 HTML, 🎨 CSS, and ⚡ JavaScript.

✨ Features
📄 Create, edit, and delete notes

📌 Pin important notes

🎨 Add background colors for better note organization

🗃️ LocalStorage support – your notes are saved even after refreshing

🔍 Search and filter notes easily

💻 Fully responsive and mobile-friendly UI
